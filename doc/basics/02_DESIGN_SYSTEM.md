# 🎨 Salati — Design System (الهوية البصرية)

> هذا الملف يحدد الهوية البصرية الكاملة للمشروع: الألوان، الخطوط، الأيقونات، والستايل العام.
> **أي عنصر UI لازم يلتزم بالنظام ده.**

---

## 📌 فلسفة التصميم

| المبدأ | التوضيح |
|--------|---------| 
| **هادئ وروحاني** | التطبيق مرتبط بالصلاة — الألوان والتصميم لازم يعكسوا الهدوء والسكينة |
| **داكن أساسي** | Dark mode كافتراضي (أريح للعين + يناسب الاستخدام الطويل) |
| **بسيط ومركّز** | المستخدم يشوف اللي يحتاجه بسرعة بدون تشتيت |
| **احترافي** | مش شكل "تطبيق طالب" — لازم يبان برنامج حقيقي |
| **Hybrid Style** | Dashboard ثابت + Settings slide panel — بدون Sidebar |

## 🕌 الأيقونات والرموز

> **قرار مهم:** نستخدم **Emoji نصية فقط** — بدون صور PNG/SVG.
> السبب: مفيش مشاكل DPI/resolution + سهلة في الكود + بتتكبر تلقائي.

### أيقونة التطبيق (الوحيدة كصورة)
- **الشكل**: مسجد مبسط (minaret/dome)
- **الألوان**: أخضر (#1B8A6B) + ذهبي (#C8A96E) على خلفية داكنة
- **الأحجام**: 16x16, 32x32, 48x48, 128x128, 256x256 (ICO)
- **الاستخدام**: app_icon.ico + tray icons

## 📐 Layout Guidelines

### النافذة الرئيسية — Full Mode (frmMain)
```
┌────────────────────────────────────────────────┐
│ ucTitleBar (📌 ⚙️ 🌙 AR — ✕)                   │
├────────────────────────────────────────────────┤
│                                                 │
│       ucNextPrayer (Hero — Countdown)           │
│                                                 │
│  [🌅 Card] [☀️ Card] [🌤️ Card] [🌇 Card] [🌙 Card] │
│                                                 │
├────────────────────────────────────────────────┤
│ ucInfoBar (📅 Date │ 📍 City │ ▼ Tray)          │
└────────────────────────────────────────────────┘

أبعاد: 700 x 500 px
بدون Sidebar — Dashboard ثابت
```

### Widget Mode
```
┌──────────────────────────────────┐
│ 🌤️ العصر 03:45  01:23:45  📌 ✕ │
└──────────────────────────────────┘

أبعاد: 300 x 85 px
Always on top
```

### نافذة التنبيه (frmAlert)
```
┌──────────────────────────────────┐
│      تبقى 10 دقائق على          │
│      صلاة العصر 🌤️               │
│        03:45 PM                  │
│  "حي على الصلاة حي على الفلاح"  │
│  [🔇 كتم]       [✅ تم]          │
└──────────────────────────────────┘

أبعاد: 420 x 300 px
TopMost, Borderless, أعلى يمين الشاشة
```

### Spacing & Padding
| العنصر | القيمة |
|--------|--------|
| **Padding (نافذة)** | 24px |
| **Padding (cards)** | 16px |
| **الفراغ بين Cards** | 12px |
| **الفراغ بين العناصر** | 8px |
| **Border Radius** | 12px (cards), 8px (buttons), 20px (pills) |

## 🎭 Animations & Transitions

| الحركة | المدة | الاستخدام |
|--------|-------|-----------|
| **Settings Slide-in** | 250ms | Panel يسلايد من اليمين |
| **Settings Slide-out** | 200ms | Panel يرجع |
| **Countdown Pulse** | 1000ms | نبض خفيف كل ثانية على العداد |
| **Card Hover** | 200ms | تكبير خفيف (scale 1.02) + تغيير حدود |
| **Alert Slide-in** | 300ms | نافذة التنبيه تدخل من الأعلى |
| **Toast** | 3000ms | ظهور ← ثبات ← اختفاء |
| **Tray Blink** | 500ms | أيقونة الـ tray تومض عند التنبيه |
| **Widget Transition** | 300ms | Full → Widget أو Widget → Full |
| **Overlay Fade** | 200ms | تعتيم الخلفية خلف Settings |
