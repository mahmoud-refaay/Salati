# 🕌 Salati — Project Overview

## المشكلة (Problem)

ناس كتير بتقعد قدام الكمبيوتر ساعات طويلة — شغل، دراسة، برمجة، أو gaming — وبتنسى مواعيد الصلاة أو بتاخد بالها إن الوقت فات. التطبيقات الموجودة على الموبايل مش كفاية لأن الموبايل ممكن يكون صامت أو بعيد.

## الحل (Solution)

**Salati** هو تطبيق ديسكتوب خفيف لـ Windows بيشتغل في الخلفية وبينبه المستخدم **قبل** وقت الصلاة. المستخدم عنده تحكم كامل في كل حاجة:
- يختار مصدر مواعيد الصلاة (API تلقائي أو إدخال يدوي)
- يحدد قبل كل صلاة بكام دقيقة يجيله التنبيه
- يفعّل أو يعطّل التنبيه لكل صلاة على حدة
- يشغّل التطبيق في 3 أوضاع: كامل / مصغّر (Widget) / خلفية (Tray)

## الجمهور المستهدف (Target Audience)

- مستخدمين Windows مسلمين (عرب وغير عرب)
- الناس اللي بتقضي وقت طويل على الكمبيوتر
- طلاب، مبرمجين، موظفين

## القيمة المضافة (Value Proposition)

| النقطة | التفاصيل |
|---|---|
| **تنبيه قبل الصلاة** | مش بس وقت الأذان — كمان قبله بالدقائق اللي المستخدم يحددها |
| **تحكم كامل** | المستخدم يتحكم في كل تفصيلة: المواعيد، الأصوات، الدقائق |
| **خفيف** | بيشتغل في الـ System Tray ومش هياثر على أداء الجهاز (~15-25 MB) |
| **3 أوضاع** | Full Dashboard + Widget (always-on-top) + Tray (خلفية) |
| **مفيش حساب** | كل حاجة محلية — مفيش تسجيل دخول ولا إنترنت إلزامي |
| **متعدد اللغات** | عربي وإنجليزي |
| **احترافي من جواه** | Registry + SQL Server + Event Viewer + Error Logging |

## النسخ المخططة (Versions Roadmap)

### v1.0 — MVP
- مواعيد صلاة (API + يدوي)
- تنبيهات مخصصة بالدقيقة
- صوت أذان + Popup
- System Tray + Widget Mode
- إعدادات في Windows Registry
- عربي + إنجليزي + ثيمان (Dark/Light)
- Error Logging (Event Viewer + DB)

### v1.5 — Enhanced
- أذكار بعد الصلاة
- أذكار الصباح والمساء
- تتبع الصلاة (هل صلّيت؟)
- أصوات أذان مخصصة

### v2.0 — Advanced
- تقويم هجري
- اتجاه القبلة
- تلاوة قرآن مدمجة

## التقنيات المستخدمة (Tech Stack)

| التقنية | الاستخدام |
|---|---|
| C# (.NET 9) | اللغة الأساسية |
| WinForms | واجهة المستخدم |
| Guna.UI2 | مكتبة UI احترافية |
| 3-Tier Architecture | DAL → BLL → UI |
| SQL Server | قاعدة البيانات (8 جداول + ~30 SP) |
| ADO.NET + Stored Procedures | الوصول للبيانات |
| Windows Registry | إعدادات المستخدم (مع fallback تلقائي) |
| Windows Event Viewer | تسجيل الأحداث والأخطاء |
| Aladhan API | مواعيد الصلاة التلقائية |
| NAudio | تشغيل أصوات الأذان |
| Git & GitHub | إدارة الكود والتعاون |

## نمط الواجهة (UI Style)

**Hybrid Style** — بسيط من بره، احترافي من جواه:
- **Dashboard ثابت** (لا Sidebar) مع Countdown + 5 كروت صلاة
- **Settings Slide Panel** يسلايد من اليمين (4 tabs)
- **Widget Mode** — strip صغير always-on-top
- **System Tray** — يشتغل في الخلفية
- **أيقونات Emoji** (🌅☀️🌤️🌇🌙) — بدون صور PNG
