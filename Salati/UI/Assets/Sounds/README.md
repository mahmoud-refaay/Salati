# Adhan Sound Files

Place your Adhan audio files (.wav / .mp3) in this folder.

## Expected Files:
- `adhan_fajr.wav`    — أذان الفجر
- `adhan_default.wav` — أذان باقي الصلوات
- `notification.wav`  — صوت التنبيه القصير

## Notes:
- WAV format is recommended for fast playback in WinForms.
- MP3 will require NAudio or similar library.
- Files should be mono or stereo, 16-bit, 44100 Hz for best compatibility.
